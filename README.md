# Devan Desai - Portfolio

Personal portfolio website showcasing my experience, projects, and skills as a Cloud Support Associate at AWS.

## About

This is a single-page portfolio website built with HTML, CSS, and vanilla JavaScript. It features a modern, dark-themed design with smooth animations and responsive layout.

## Features

- Responsive design that works on desktop, tablet, and mobile
- Smooth scroll navigation
- Typing animation effect
- Project showcase
- Experience timeline
- Skills and certifications display
- Contact section with social links

## Sections

- **Hero**: Introduction with animated typing effect
- **About**: Personal background and interests
- **Experience**: Professional work history
- **Education**: Academic background
- **Certifications**: AWS and other professional certifications
- **Projects**: Portfolio of completed projects
- **Contact**: Links to LinkedIn, GitHub, and resume

## Technologies Used

- HTML5
- CSS3 (with custom properties and animations)
- JavaScript
- Google Fonts (Instrument Serif, DM Sans, JetBrains Mono)
- Remix Icon
